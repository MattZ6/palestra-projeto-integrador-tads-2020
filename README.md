### 📝 Como usar?

```bash
# Clone o repositório
$ git clone https://github.com/MattZ6/palestra-projeto-integrador-tads-2020.git viagenzinhas

# Adentre a pasta do projeto
$ cd viagenzinhas

# Abra o arquivo index.html no seu navegador
$ x-www-browser index.html
```
